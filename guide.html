Setup

Connect your API key

Follow these steps once — then the extension is ready to use on any USACO problem.

01
Step 1
Open Google AI Studio

Sign into AI Studio with your Google account. It's free — no paid plan or credit card required.

🔒 aistudio.google.com/welcome →
02
Step 2
Go to the API Keys page

Navigate directly to the API keys section. Accept the Terms of Service if prompted — this is a one-time step.

🔒 aistudio.google.com/api-keys →
03
Step 3
Create and copy your key

Click + Create API key, give it a name, then copy the key — you won't be able to view it again after closing the dialog.

Usage

How the extension works

Once your key is saved, AI Coach activates automatically on any USACO problem page that has an available editorial.

⬡
FAB Toggle Button

A floating button appears at the bottom-left of the page whenever an editorial is available for the current problem.

◫
Sidebar Chat Panel

Click the FAB to open a full-height sidebar. The editorial is loaded automatically as context — no copy-pasting needed.

◎
Ask Anything

Type your question — "What's the main idea?", "Give me a hint", "Explain the time complexity" — and get a guided answer that won't just spoil the solution.

◈
Difficulty Ratings

A color-coded difficulty badge is injected next to each problem title, sourced from community ratings — so you know what you're tackling before you start.
